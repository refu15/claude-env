# 補助金申請ナレッジベース

## 概要
TWIST株式会社の補助金申請サポート事業における実務知識を体系化したナレッジベース。
申請支援の全プロセス（ヒアリング→書類収集→申請書作成→提出）をカバーする。

## 対象者
- G2（案件担当・ヒアリング・申請支援）: 櫻井、小野寺、河合
- Admin（申請・報告事務）: 今野、清水
- M（進捗管理）: 伊藤、河合

## 案件ステータスフロー
```
未接触
  → 申請意向確認中
  → 初回打合せ済
  → 申請書類作成中
  → 申請完了
  → 採択 / 不採択
  → 実績報告中
  → 入金待ち
  → 完了 / キャンセル
```
