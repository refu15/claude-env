# 小規模事業者持続化補助金（第19回）

## 基本情報
- **正式名称**: 小規模事業者持続化補助金＜一般型・通常枠＞
- **管轄**: 日本商工会議所（商工会議所地区）/ 全国商工会連合会（商工会地区）
- **公募期間**: 2026/3/6〜2026/4/30 17:00
- **様式4発行受付締切**: 2026/4/16（申請締切の2週間前）
- **採択発表**: 2026年7月頃
- **事業実施期間**: 交付決定日〜2027/6/30
- **TWIST報酬区分**: Mid（20〜50万円）

## 補助率・上限額

| 枠・特例 | 上限額 | 補助率 |
|---------|--------|--------|
| 通常枠（基本） | 50万円 | 2/3 |
| インボイス特例（上乗せ） | +50万円（計100万円） | 2/3 |
| 賃金引上げ特例（上乗せ） | +150万円（計200万円） | 2/3（赤字3/4） |
| 両特例併用 | 最大250万円 | 2/3（赤字3/4） |
| 創業型（第3回） | 200万円（特例で最大250万円） | 2/3 |

## 申請要件
### 業種別の従業員数上限
| 業種 | 従業員数 |
|------|---------|
| 商業・サービス業（宿泊業・娯楽業除く） | 5人以下 |
| 製造業・その他（宿泊業・娯楽業含む） | 20人以下 |

### 除外要件
- 資本金5億円以上の企業に100%出資されている法人
- 直近3年間の課税所得平均が15億円超
- 前回採択から10ヶ月以内の重複申請
- 医師、歯科医師、助産師、学校法人、宗教法人、NPO法人等

## 対象経費（費目）
1. 機械装置等費（汎用性高いもの不可）
2. 広報費（チラシ・看板・広告等）
3. ウェブサイト関連費（**上限: 1/4かつ最大50万円、単独申請不可**）
4. 展示会等出展費
5. 旅費
6. 新商品開発費
7. 資料購入費
8. 借料
9. 設備処分費（対象経費の1/2以内）
10. 委託・外注費（HP制作はウェブサイト関連費に計上）
11. 雑役務費（補助事業専従のアルバイト人件費）

## 申請フロー
1. GビズIDプライム取得（3〜4週間）
2. 公募要領熟読
3. 経営計画書（様式2）・補助事業計画書（様式3）作成
4. 商工会議所で様式4発行依頼【4/16締切】
5. jGrantsで電子申請【4/30 17:00締切】
6. 採択審査→発表（7月頃）
7. 交付決定→補助事業実施
8. 実績報告（2027/7/10）
9. 補助金受領

## 審査の加点項目
**重点政策加点（最大1種）**: 赤字賃上げ、事業環境変化（第19回新設）、東日本大震災、能登半島地震、くるみん・えるぼし
**政策加点（最大2種）**: 賃上げ、経営力向上計画、事業承継、過疎地域、一般事業主行動計画、パワーアップ型

## 経営計画書のポイント
1. SWOT分析が具体的か
2. 課題→対策→効果のストーリーが一貫しているか
3. 補助金なしでは実施できない理由があるか
4. 数値目標が具体的か
5. 事業終了後も効果が持続するか
- 計画書は合計**8枚程度**、図表・グラフを効果的に活用
