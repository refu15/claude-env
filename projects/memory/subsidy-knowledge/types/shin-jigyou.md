# 新事業進出補助金

## 概要・目的
中小企業・小規模事業者が既存事業の強みを活かして新たな事業分野に進出する取組（新市場参入・新製品・新サービス開発等）に必要な経費を支援する。

- **規模分類**: Large
- **成果報酬**: 50万円〜

## 申請フロー
1. **事業アイデアの整理**: 新事業の概要・既存事業との関連性・市場性をヒアリング
2. **市場分析**: TAM/SAM/SOM、競合状況、参入の優位性を整理
3. **収支計画策定**: 3〜5年の売上・費用・利益の計画を作成
4. **事業計画書作成**: 「なぜ今・なぜ自社か」を中心に記述
5. **書類収集・申請**
6. **採択後**: 事業実施→実績報告→補助金受領

## 事業計画書で重視される観点
- **新規性**: 既存事業とは明確に異なる事業であること
- **市場性**: 参入市場の規模・成長性・競合状況の分析
- **実現可能性**: 自社の強み・経営資源と新事業のフィット感
- **収益性**: 投資回収見込みが明確であること

## よくある不備・キャンセル理由
- 新規性が不明確（既存事業の延長と判断される）
- 市場分析が感覚的で、データに基づいていない
- 収支計画の根拠が薄い（売上目標が楽観的すぎる）
- 既存事業との相乗効果の説明が不足
