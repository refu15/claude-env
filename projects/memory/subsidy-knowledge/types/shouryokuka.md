# 省力化補助金（中小企業省力化投資補助金）

## 概要・目的
人手不足に悩む中小企業・小規模事業者が、IoT・ロボット等の汎用製品を導入して省力化・生産性向上を実現するための投資を支援する。
カタログに掲載された製品の中から選択して申請する「カタログ型」が特徴。

- **規模分類**: Large
- **成果報酬**: 50万円〜

## 対象者要件
- 中小企業者・小規模事業者（中小企業基本法に基づく定義）
- 労働生産性を向上させる取組であること
- 補助事業終了後3年間の事業化状況報告ができること

### 主な対象外
- 同一事業・設備で他の補助金を受給予定の場合
- 一般社団法人・公益法人等（条件による）

## 補助率・補助上限額
| 従業員規模 | 補助率 | 補助上限額 |
|-----------|--------|----------|
| 5人以下 | 1/2 | 200万円 |
| 6〜20人 | 1/2 | 500万円 |
| 21人以上 | 1/2 | 1,000万円 |

賃上げ要件を満たす場合は上限額が1.5倍（最大1,500万円）に拡大される場合あり。

## 必要書類チェックリスト
### 必須書類
- [ ] 申請書（GビズID必要）
- [ ] 直近1〜2期分の決算書（貸借対照表・損益計算書）
- [ ] 法人税・消費税の納税証明書
- [ ] 見積書（カタログ掲載製品の販売代理店から取得）
- [ ] 登記簿謄本（法人の場合）
- [ ] 従業員数証明書類

### 追加書類（条件による）
- [ ] 賃上げ計画を示す書類（賃上げ加点を受ける場合）
- [ ] 製品カタログ・仕様書

## 申請フロー
1. **要件確認**: 従業員数・業種・導入設備がカタログ対象かを確認
2. **省力化効果のヒアリング**: 現在の人手不足状況・導入後の削減工数を具体的に数値化
3. **製品選定**: カタログから対象製品を選び、販売代理店に見積依頼
4. **申請書作成**: 省力化効果を具体的に記述（導入前後の工数比較等）
5. **GビズIDでの電子申請**
6. **採択後**: 製品購入・設置→実績報告→補助金受領

## よくある不備・キャンセル理由
- 導入製品がカタログ未掲載（申請前の確認不足）
- 省力化効果の数値根拠が不明確（削減工数・コストの試算がない）
- 見積書が販売代理店以外からのもの
- GビズID未取得で締切に間に合わない
