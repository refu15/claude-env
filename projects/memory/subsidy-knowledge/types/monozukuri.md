# ものづくり補助金（ものづくり・商業・サービス生産性向上促進補助金）

## 概要・目的
中小企業・小規模事業者が取り組む革新的な製品・サービス開発、生産プロセス・サービス提供方法の改善に必要な設備投資等を支援する。

- **規模分類**: Large
- **成果報酬**: 50万円〜

## 対象者要件
- 中小企業者（製造業・建設業・運輸業その他: 資本金3億円以下 または 常勤従業員300人以下等）
- 小規模事業者
- 特定非営利活動法人（条件あり）
- 3〜5年の事業計画を策定し、付加価値額・給与支給総額・事業場内最低賃金の目標達成にコミットすること

### 申請要件（直近）
- 事業計画期間中に付加価値額を年率平均3%以上増加させる計画
- 給与支給総額を年率平均1.5%以上増加させる計画（又は事業場内最低賃金を地域別最低賃金+30円以上の水準にする）

## 申請フロー
1. **要件・枠の確認**: 事業内容・従業員数・投資規模で最適枠を選定
2. **事業計画書の骨子作成**: 現状→課題→取組内容→効果の流れで設計
3. **詳細ヒアリング**: 投資設備・経費詳細・KPI目標値を確認
4. **見積取得**: 設備メーカー・販売店から見積書を取得
5. **事業計画書の完成版作成**
6. **電子申請（GビズID必要）**
7. **採択後**: 交付申請→事業実施→実績報告→確定検査→補助金受領

> 注意: GビズIDのプライム取得に2〜4週間かかる場合あり。早期に確認すること

## よくある不備・キャンセル理由
- 事業計画書の革新性・独自性の説明が不足（既存製品・サービスの改良と区別できない）
- 付加価値額・給与増加の目標が根拠なく設定されている
- 見積書が1社のみで複数社からの見積比較がない
- GビズID未取得で締切に間に合わない
