# ヒアリング

## 概要
初回打合せ・ヒアリングで収集すべき情報の一覧とコツ。
ヒアリングシート（事業者情報ヒアリングシート+必要書類リスト.xlsx）に基づく。

## 対象者
G2（案件担当）: 初回打合せ・ヒアリング担当者

## ヒアリング項目一覧

### A. 会社概要
| 項目 | 記載例・メモ |
|------|------------|
| 法人名（正式名称） | 株式会社〇〇 |
| 代表者名 | 氏名・読み仮名 |
| 所在地（本社） | 〒000-0000 都道府県〜 |
| 電話番号 | 代表・担当者携帯 |
| 資本金 | 〇〇万円 |
| 株主構成 | 代表〇〇%、その他〇〇% |
| 設立年月日 | 〇年〇月〇日（創業年と設立年が異なる場合は両方） |
| 従業員数 | 正社員〇人・パート〇人（補助金の規模要件に直結） |
| 事業概要（業種）| 〇〇業（日本標準産業分類の中分類まで把握） |
| 公式WEBサイト | URL |

### B. 会社・事業紹介（100〜500字）
- 何をやっている会社か、強み・特徴は何か
- 主な顧客層・販売チャネル
- 創業の経緯や代表のビジョン
- 直近の業績トレンド（成長中か・横ばいか・課題があるか）

> ポイント: 申請書の「事業者概要」セクションにそのまま使える内容を引き出す

### C. 経営者経歴
- 代表者の職歴・業界経験年数
- 過去の実績・受賞歴・メディア掲載等
- 前職から現事業への流れ（なぜこの事業を始めたか）

> ポイント: 採択審査で「人への信頼」も見られる。経歴の強みを引き出す

### D. 補助金対象取組の内容
| 項目 | 確認ポイント |
|------|------------|
| 取組の概要 | 何をしたいか・何を購入・導入したいか |
| 発注先（予定） | 機器メーカー・ベンダー・工事業者等の名称 |
| 発注金額（予定）| 税込・税抜の別、概算でも可 |
| 取組の目的 | 売上増加・コスト削減・省人化・新市場参入等 |
| 課題の背景 | なぜ今この取組が必要か |
| 期待される効果 | 具体的な数値目標（売上〇%増、作業時間〇時間削減等）|

### E. 財務・税務状況の確認
- 直近の決算状況（黒字か赤字か）
- 税金の滞納がないか（補助金要件に関わる）
- 他の補助金の申請・受給状況（重複申請の確認）

> 注意: 滞納がある場合は申請要件を満たさない可能性がある。早期確認が必須

### F. スケジュール確認
- 申請したい補助金の締切
- 設備購入・工事の予定時期
- 決算書等の準備に要する時間

## ヒアリングのコツ・ポイント

### 場の設定
- 初回は「補助金の説明」より「事業の話を聞く」姿勢で臨む
- メモより会話を優先し、後でシートに転記する
- オンライン実施の場合はカメラONを推奨（信頼関係構築のため）

### 情報の引き出し方
- 「従業員は何人ですか？」ではなく「現場の方は何人くらいいらっしゃいますか？」と聞くと答えやすい
- 経費・金額は「おおよそで構わないので」と前置きして聞く
- 取組内容は「具体的にどんな機械（システム）を入れたいですか？」と物・事から聞く

### 要件チェックの優先順位
1. **従業員数**: 補助金の規模要件に最も直結するため最初に確認
2. **税金滞納の有無**: 不採択・取消リスクを早期に排除
3. **他補助金との重複**: 同一経費での二重申請は禁止

### よくある引き出しにくい情報と対処法
| 情報 | 引き出しにくい理由 | 対処法 |
|------|-----------------|--------|
| 株主構成 | プライベートな感覚がある | 「補助金の書類で必要になるので」と理由を説明 |
| 資本金 | 知らない経営者もいる | 「登記簿謄本に記載があります」と案内 |
| 正確な従業員数 | パート・アルバイトを含むか曖昧 | 「正社員・パートそれぞれ何人か」と分けて聞く |
| 決算の状況 | 赤字の場合は答えたくない | 「書類で確認しますので大丈夫です」と安心させる |

## ヒアリング後のアクション
1. ヒアリングシートに情報を転記・整理
2. 申請可能な補助金と最適な枠を選定
3. 必要書類チェックリストをカスタマイズして事業者に送付
4. ヒアリング不足の情報があれば追加確認
