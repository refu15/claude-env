# 必要書類チェックリスト

## 概要
補助金申請に必要な書類の全体像と、補助金種別ごとの追加書類、よくある不備パターンをまとめたリファレンス。
書類収集は「申請書類作成中」ステータスの最重要タスク。

## 共通必要書類（ほぼ全ての補助金で必要）

### 法人の場合
| 書類 | 取得方法 | 注意点 |
|------|---------|--------|
| 履歴事項全部証明書（登記簿謄本） | 法務局・オンライン申請 | 発行から3ヶ月以内のもの |
| 決算書一式（貸借対照表・損益計算書） | 自社保管（税理士から取得）| 指定期数（1〜3期）を確認 |
| 法人税申告書（表紙・別表一）| 自社保管 | 決算書と合わせて提出 |
| 納税証明書 | 税務署・e-Tax | **種類に要注意（後述）** |
| 見積書 | 発注予定先から取得 | 補助対象経費ごとに1通以上 |

### 個人事業主の場合
| 書類 | 取得方法 | 注意点 |
|------|---------|--------|
| 確定申告書（第一表・第二表） | 自社保管・税務署 | 指定期数を確認 |
| 収支内訳書または青色申告決算書 | 自社保管 | |
| 納税証明書 | 税務署・e-Tax | |
| 開業届（創業まもない場合）| 税務署提出済みのもの | |

## 納税証明書の種類（重要）

補助金によって「納税証明書その〇」の指定が異なる。誤った種類を取得すると差し戻しになる。

| 種類 | 記載内容 | 主な用途 |
|------|---------|---------|
| その1 | 納税額の証明 | 法人税・所得税の額を証明 |
| その2 | 未納税額のないことの証明 | **最もよく使われる（滞納がないことを証明）** |
| その3 | 未納の税額がないことの証明（法人税・消費税）| ものづくり補助金等で指定 |
| その3の3 | 未納の税額がないことの証明（所得税・消費税）| 個人事業主向け |

> ルール: 不明な場合は「その2（法人税）とその2（消費税）の両方」を取得しておくと安全

## 補助金種別ごとの追加書類

### 再起支援補助金
- 事業困難を証明する書類（売上減少の試算表・被災証明等）
- 再起に向けた事業計画の詳細

### 小規模事業者持続化補助金
- 商工会議所・商工会の支援確認書（事前相談が必要）
- 経営計画書・補助事業計画書（所定様式）

### ものづくり補助金
- GビズIDのログイン情報（電子申請必須）
- 設備の仕様書・カタログ
- 賃金引上げ計画の誓約書

### 省力化補助金
- カタログ掲載製品の確認資料
- 省力化効果の試算書（導入前後の工数比較）

### 新事業進出補助金
- 市場調査データ・業界レポート
- 顧客ニーズを示す資料

## 見積書の要件
- **金額**: 補助金種別ごとに複数見積が必要な金額閾値が異なる（税抜50万円・100万円等）
- **記載必須事項**: 会社名・日付・品名・数量・単価・合計額・有効期限
- **有効期限**: 申請時点で有効なものであること（期限切れは差し戻し）
- **税込・税抜の明記**: 補助対象は税抜金額のため、消費税を分けて記載

> よくある不備: 「一式〇〇円」という記載で品名・数量が不明なもの。必ず詳細を記載してもらうこと

## 書類収集の進め方

### 優先順位
1. **納税証明書**: 税務署・e-Taxで取得（即日〜1週間）
2. **登記簿謄本**: オンライン申請なら3〜5営業日
3. **見積書**: 発注先へ依頼（1〜2週間かかることも）
4. **決算書**: 税理士から取得（事前に依頼しておく）

### 締切から逆算したスケジュール
| 締切まで | すべきこと |
|---------|----------|
| 4週間以上 | 全書類の収集開始 |
| 2〜3週間 | 見積書・証明書の催促 |
| 1週間 | 書類の最終確認・申請書と照合 |
| 3日前 | 電子申請のテスト・最終送信準備 |

## よくある不備パターン

| 不備内容 | 発生頻度 | 防止策 |
|---------|---------|-------|
| 納税証明書の種類が誤り | 高 | 申請前に公募要領で指定の種類を確認 |
| 登記簿謄本の有効期限切れ | 高 | 申請直前（3ヶ月以内）に取得 |
| 見積書の記載不足（品名・単価不明）| 高 | 見積依頼時に「品名・数量・単価を明記」と指示 |
| 決算書の期数不足 | 中 | 公募要領の「提出書類一覧」を事前に確認 |
| 見積書の有効期限切れ | 中 | 取得後すぐに申請準備を進める |
| 株主構成の確認漏れ（みなし大企業の問題）| 中 | ヒアリング時に株主構成を必ず確認 |
| 申請書の数値と決算書の数値の不一致 | 中 | 提出前に申請書と添付書類を突合確認 |

> 「みなし大企業」に注意: 大企業（資本金3億円超等）が3分の1以上出資している場合は中小企業として申請できない
