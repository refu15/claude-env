# 補助金業務 Google Drive フォルダ構造

## 標準フォルダ構造

```
補助金業務/
├── 案件/
│   └── {案件番号}_{事業者名}/     ← 案件ごとに作成
│       ├── 01_公募要領/           ← 公募要領PDF等
│       ├── 02_ヒアリング/         ← ヒアリングシート、顧客資料
│       ├── 03_申請書類/           ← 申請書、事業計画書
│       ├── 04_見積書・発注書/     ← 見積書、発注先情報
│       ├── 05_証明書類/           ← 納税証明書、謄本、決算書
│       ├── 06_申請完了記録/       ← 申請完了スクショ、受付番号
│       └── 07_実績報告/           ← 実績報告書類
├── テンプレート/                   ← 共通テンプレート
│   ├── ヒアリングシート/
│   ├── 申請書テンプレート/
│   └── チェックリスト/
├── 公募情報/                       ← 補助金種別ごとの公募要領
│   ├── 再起支援/
│   ├── 持続化/
│   ├── ものづくり/
│   ├── 省力化/
│   └── 新事業進出/
└── ナレッジ/                       ← 過去事例・ノウハウ
    ├── 採択事例/
    └── 不採択分析/
```
