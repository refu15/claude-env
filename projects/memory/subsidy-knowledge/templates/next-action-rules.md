# 次アクション判定ルール

## 概要
案件のステータスと状況に応じて、次に取るべきアクションを判定するためのルール表。
案件管理シートのステータスと照合して使用する。

## 対象者
G2（案件担当）、M（進捗管理）、Admin

## ステータス別次アクション一覧

| ステータス | 条件 | 推奨アクション | 担当 | 優先度 |
|-----------|------|--------------|------|--------|
| 未接触 | — | 初回コンタクト（LINE/電話）| G1 | 通常 |
| 申請意向確認中 | 3日無反応 | フォロー連絡（LINE）| G1/G2 | 通常 |
| 申請意向確認中 | 7日無反応 | 電話でのフォロー + 状況確認 | G1/G2 | 高 |
| 初回打合せ済 | ヒアリングシート未送付 | ヒアリングシートを送付 | G2 | 高 |
| 初回打合せ済 | ヒアリングシート送付済 + 3日無反応 | 記入催促連絡 | G2 | 高 |
| 初回打合せ済 | ヒアリングシート送付済 + 7日無反応 | 電話にて催促・期限を設ける | G2 | 緊急 |
| 申請書類作成中 | 納税証明書未取得 | 取得依頼 + 取得方法の案内（税務署・e-Tax） | G2/Admin | 高 |

| 申請書類作成中 | 申請書ドラフト完成 | 事業者に確認依頼 | G2 | 高 |

| キャンセル | — | キャンセル理由を案件管理シートに記録 | M | — |

## 優先度の定義
| 優先度 | 対応期限の目安 | 内容 |
|--------|-------------|------|
| 緊急 | 当日〜翌日 | 締切・リスクに直結する対応 |
| 高 | 2〜3営業日以内 | 進捗に影響する重要な対応 |
| 通常 | 1週間以内 | 定常的なフォロー |

## フォロー連絡の文例（LINE）

### ヒアリングシート記入催促
```
お世話になっております、TWIST株式会社の〇〇です。
先日お送りしたヒアリングシートのご記入はいかがでしょうか？
申請期限の関係もあり、〇月〇日（〇）までにご返送いただけますと助かります。
ご不明点があればお気軽にご連絡ください！
```

### 書類取得催促（納税証明書）
```
お世話になっております、TWIST株式会社の〇〇です。
申請書類の確認をしておりますが、納税証明書がまだ未着となっております。
税務署またはe-Taxで取得できますので、お早めにご手配をお願いいたします。
取得方法のご案内もできますのでお気軽にお声がけください。
```

### 不採択後の次回申請提案
```
お世話になっております、TWIST株式会社の〇〇です。
このたびの〇〇補助金の審査結果についてご連絡いたします。
残念ながら今回は不採択という結果となりました。
審査のフィードバック内容を踏まえ、次回公募での再申請に向けた改善策をご提案できればと思います。
よろしければ一度お打合せの機会をいただけますでしょうか？
```

## 案件ステータスの変更判断基準

| 変更前 | 変更後 | 変更タイミング |
|-------|-------|-------------|
| 未接触 | 申請意向確認中 | 初回コンタクト完了後 |
| 申請意向確認中 | 初回打合せ済 | 打合せ（対面/オンライン）完了後 |
| 申請意向確認中 | キャンセル | 申請しないと意思表示があった場合 |
| 初回打合せ済 | 申請書類作成中 | ヒアリングシート回収・書類収集開始後 |
| 申請書類作成中 | 申請完了 | 補助金事務局へ申請書を提出した後 |
| 申請完了 | 採択 | 採択通知を受領後 |
| 申請完了 | 不採択 | 不採択通知を受領後 |
| 採択 | 実績報告中 | 補助事業実施後・実績報告手続き開始後 |
| 実績報告中 | 入金待ち | 実績報告が受理された後 |
| 入金待ち | 完了 | 補助金が入金されたことを確認後 |
