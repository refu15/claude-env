# 業種別ライティングサンプル・ポイント

最終更新: 2026-03-25
作成者: knowledge-builder
根拠: LEIMEI（広告代理店）、まりぼう（居酒屋）の採択事例分析

---

## 概要

持続化補助金の採択事例を業種別に整理した実例集。
各業種ごとに「強みの見つけ方」「市場データの探し方」「よくある課題と取組の対応表」を掲載。

---

## 業種1: 広告・マーケティング業（LEIMEI事例）

### 事業概要

- **業態**: Web専門の成果報酬型広告代理店
- **補助事業**: HP改修・営業資料作成による新規顧客開拓
- **採択ポイント**: 具体的な競合分析と差別化2軸の明確化

---

### この業種特有の強みの見つけ方

**差別化ポイントを2軸で整理する（LEIMEI方式）:**

| 軸 | 内容 | LEIMEIの例 |
|----|------|-----------|
| 対応範囲の広さ | 一気通貫 vs 部分対応 | 企画・制作・運用を社内完結 |
| 専門性の深さ | 特定媒体 vs 全般対応 | SNS全媒体（Instagram・X・TikTok・YouTube）に対応 |

**このパターンが使える業種:** コンサル、Web制作、IT、デザイン、会計

---

### 市場データの探し方

**広告・マーケティング業で使えるデータ:**

| データ | 出典 | 発表時期 |
|--------|------|---------|
| 日本の総広告費 | 電通「日本の広告費」 | 毎年2月 |
| インターネット広告費 | 電通「日本の広告費」内 | 毎年2月 |
| 中小企業のDX推進状況 | 中小企業庁「中小企業白書」 | 毎年4〜5月 |
| SNS利用率 | 総務省「情報通信白書」 | 毎年7月 |

**地域データの探し方:**
- 商工会議所の調査レポート（地域中小企業のWeb活用状況など）
- 都道府県産業振興センターの調査

---

### 経営計画（Table 7）の書き方ポイント

#### 1-1. 事業概要

**機能・サービス主体の書き方（感情より論理）:**
```
当社は○年創業のWeb専門広告代理店です。
主要サービスは[サービス1]・[サービス2]・[サービス3]で、
中小企業・スタートアップ向けに成果報酬型モデルで提供しています。
従業員○名、直近年度売上○○万円。
```

#### 1-3. 経営課題

**広告・IT系によくある課題パターン:**
- 「既存顧客からの紹介のみで新規開拓ができていない」
- 「Webサイトが古く、初めて訪れた顧客への訴求力が不足している」
- 「営業資料がなく口頭説明のみで、商談成約率が低い」
- 「競合他社との差別化が顧客に伝わっていない」

#### 2-1. 市場動向

**マクロ→ミクロの展開例:**
```
電通の調査によると、2024年の日本の総広告費は7兆6,730億円に達し、
特にインターネット広告費は○兆円と全体の○%を占め、年々拡大しています。
一方、中小企業庁の調査では、中小企業のデジタル広告活用率は○%にとどまり、
潜在的な需要は依然として大きいと考えられます。
○○市においても、商工会議所の調査で○%の企業が「Web集客を強化したい」
と回答しており、当社へのニーズは高まっています。
```

#### 3. 強み・弱み

**広告・IT業種のSWOT例:**

強み:
```
① 一気通貫対応: 企画・制作・運用を社内完結。外注コストゼロ・
   クイックターンアラウンドを実現（業界平均の○倍のスピード）
② SNS全媒体対応: Instagram・X・TikTok・YouTube全てに対応できる
   体制は地域内で数少ない
③ 成果報酬型: 初期リスクが低いため中小企業に採用されやすい
```

弱み:
```
① Webサイトが情報不足・古いデザインで、初回訪問者への訴求力が低い
② 営業資料がなく、口頭説明のみでは実績・サービス内容が伝わりにくい
```

### 補助事業計画（Table 8）の書き方ポイント

**取組内容の具体化例:**

HP改修:
```
現在のWebサイトをリニューアルします。主な改修内容は以下の通りです。
① TOPページのリデザイン（実績・強みを視覚的に訴求）
② サービス紹介ページの新設（料金体系・成果報酬の仕組みを明示）
③ 導入事例・お客様の声ページの追加（信頼性向上）
④ お問い合わせフォームの改善（CVR向上）
制作はA社（見積額○○万円）に委託します。
```

営業資料:
```
現在口頭説明のみで行っている商談を支援する営業資料（PDF、○ページ）を作成します。
内容: 会社概要・サービス概要・料金体系・成果事例・FAQ
制作はB社（見積額○○万円）に委託します。
```

**効果の試算例:**

```
現在の商談成約率: ○%（営業資料なし・口頭のみ）
資料作成後の想定成約率: ○%（同業他社平均を参考）

月間新規商談数: ○件
月間新規成約件数: ○件 × ○% = ○件（現在○件→○件へ増加）
平均案件単価: ○万円/月
月間売上増加: +○万円
年間売上増加: +○○万円
```

---

### スケジュール例

| 時期 | 取組内容 |
|------|---------|
| 1〜2ヶ月目 | 制作業者選定・要件定義・契約 |
| 2〜4ヶ月目 | HP改修制作・営業資料制作 |
| 4ヶ月目 | 公開・完成 |
| 5ヶ月目以降 | 運用・効果測定・改善 |

## 業種2: 飲食業（まりぼう事例）

### 事業概要

- **業態**: 多国籍料理の居酒屋
- **補助事業**: 天童市繁華街への移転＋内装工事
- **採択ポイント**: 創業ストーリー、顧客の声の引用、フェルミ推定による効果試算

---

### この業種特有の強みの見つけ方

**飲食業の差別化ポイント整理:**

| 軸 | 確認すること |
|----|------------|
| メニュー | 他店にない料理・食材・調理法はあるか |
| 経験・技術 | 代表者のバックグラウンド・習得した技術 |
| 仕入れ | 独自の仕入れルート・産地直送・希少食材 |
| 雰囲気・コンセプト | 内装・世界観・ターゲット層との一致 |
| 価格帯 | 地域相場との比較 |
| 立地 | アクセス・視認性・駐車場 |

---

### 市場データの探し方

**飲食業で使えるデータ:**

| データ | 出典 | 発表時期 |
|--------|------|---------|
| 外食産業市場規模 | 一般社団法人日本フードサービス協会 | 毎年3月頃 |
| 家計の食費支出 | 総務省「家計調査」 | 毎月 |
| 飲食店の開廃業率 | 厚生労働省「衛生行政報告例」 | 毎年 |
| 訪日外国人の飲食消費 | 観光庁「訪日外国人消費動向調査」 | 四半期 |

**地域データ:**
- 市区町村の「観光統計」「人口統計」
- 商工会議所の飲食業実態調査
- 地元の人流データ（Yahoo!人流データなど）

---

### 経営計画（Table 7）の書き方ポイント

#### 1-1. 事業概要（ナラティブ型を推奨）

**物語型の書き方例:**
```
代表○○は、20代の頃バックパッカーとして東南アジア7カ国を旅し、
現地の家庭料理の奥深さに魅了されました。帰国後、独学で各国料理を習得し、
○年に天童市にて「まりぼう」を開業。
現在は多国籍料理を中心に、地元食材を使ったオリジナルメニューを提供しています。
席数20席、スタッフ○名（うちパート○名）で運営中です。
```

#### 1-2. 売上の状況

**売上推移の書き方:**
```
売上高は○年度○○万円、○年度○○万円（前年比5.8%増）、○年度○○万円（前年比16.8%増）と
コロナ禍を経て回復傾向にあります。
客単価は4,000円、月間来客数は平均○名です。
しかしながら現店舗の立地条件から新規顧客の獲得に課題があり、
さらなる売上向上のために移転を検討しています。
```

#### 2-2. 顧客ニーズ

**顧客の声を引用する型:**
```
常連のお客様からは「いつも通っているが、初めて連れてくる友人に場所を説明するのが難しい」
「もっと広い席でグループで来たい」といったご意見を複数いただいています。
また、SNS（Instagram）では「駐車場から見つけにくい」とのコメントも散見されます。
これらの声から、①アクセス・視認性の改善 ②席数の拡大 というニーズが
特に強いことが分かります。
```

### 補助事業計画（Table 8）の書き方ポイント

**移転・内装工事の取組内容記載例:**

```
【移転先の概要】
移転先: 天童市○○町○丁目（現在地から○m）
物件規模: ○m²（現在○m²より広い）
席数: 25席（現在20席）
立地の優位性: ○○通りに面し、1日の通行量○○人（○○市調査）

【工事内容】
施工業者: ○○建設株式会社（地元施工業者、見積額○○万円）
工事内容:
① 内装デザイン: 多国籍料理の世界観を表現したインテリア（○○テイスト）
② 厨房設備の移設・増設
③ 看板・ファサードの整備（視認性向上）
④ トイレ・バリアフリー対応

【工事期間中の集客維持策】
工事進捗をInstagramで発信し（フォロワー○名）、
オープン前から期待感を醸成します。
```

**フェルミ推定による効果試算例:**

```
【移転後の売上シミュレーション】

前提条件:
・席数: 25席（現在20席）
・回転率: 1.5回転/日（現在と同水準）
・稼働率: 30%（移転後当初）→ 50%（1年後安定後）
・客単価: 4,500円（現在4,000円。立地改善で客層向上を想定）

移転直後（安定前）:
1日来客数 = 25席 × 1.5回転 × 30% = 11.25人 ≒ 11.3人
1日売上 = 11.3人 × 4,500円 = 50,850円
年間売上 = 50,850円 × 360日 × 0.7（季節変動調整）
        ≒ 1,280万円

1年後（安定後）:
1日来客数 = 25席 × 1.5回転 × 50% = 18.75人 ≒ 18.8人
1日売上 = 18.8人 × 4,500円 = 84,600円
年間売上 = 84,600円 × 360日 × 0.7（季節変動調整）
        ≒ 2,131万円

目標: 補助事業完了後1年以内に年間売上1,500万円（現状比○%増）を達成します。
```

## 業種別のよくある課題と補助事業の対応表

| 業種 | よくある経営課題 | 適した補助事業 |
|------|---------------|-------------|
| 飲食業 | 立地が悪く新規客が来ない | 移転・内装工事 |
| 飲食業 | メニューが伝わらない | メニュー表・HP制作 |
| 飲食業 | 客単価が低い | 内装グレードアップ・新メニュー開発 |
| 小売業 | ネット販売ができていない | ECサイト構築 |
| 小売業 | 店舗の認知度が低い | 看板・HP・SNS広告 |
| サービス業 | 新規顧客開拓ができていない | HP改修・営業資料・展示会出展 |
| 製造業 | 販路が限られている | 展示会出展・カタログ作成 |
| 士業・コンサル | 信頼性の訴求が難しい | HP改修・ホワイトペーパー作成 |

## 業種横断: 採択事例から学ぶ共通パターン

### パターン1: 「地域の○○」ポジショニング

**地域で唯一・最初・最多であることを強調する。**

例:
- 「○○市で唯一の多国籍料理店」
- 「○○県内で数少ないSNS全媒体対応の広告代理店」
- 「○○地区で初のオーガニック食材専門レストラン」

### パターン2: 代表者の属人的強みを事業の強みに転換

**「代表者のこだわり」→「顧客へのメリット」に変換する。**

```
[代表者の経験・技術・こだわり]
  ↓（だから）
[他店・他社にはできないサービス/商品を提供できる]
  ↓（その結果）
[顧客が得られる具体的なメリット]
```

### パターン3: SNS・デジタルとの組み合わせ

補助事業にSNS活用を組み込むと評価が高まる傾向がある（審査員がデジタル化を重視）。

例（まりぼう）:
> 「工事期間中はInstagramで進捗を発信し、オープン前から認知度向上を図る」
