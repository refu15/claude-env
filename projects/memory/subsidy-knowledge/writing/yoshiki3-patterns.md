# 様式3（補助事業計画書② 経費明細表・資金調達方法）パターン分析

最終更新: 2026-03-25
根拠: コナミル（カフェ）、Daisies（飲食）、S-LAB（鍼灸整骨院）、まりぼう（居酒屋）の採択事例分析

---

## 概要

様式3は「経費明細表（Ⅱ）」と「資金調達方法（Ⅲ）」の2部構成。
公式Excelテンプレートに入力する形式で、計算式が組み込まれている。

---

## Ⅱ. 経費明細表の構造

### 入力項目（1行＝1経費項目）

| 列 | 内容 | 記入例 |
|----|------|--------|
| 経費区分 | ①〜⑧の費目番号 | ①機械装置等費 |
| 内容・必要理由 | 何を買うか＋なぜ必要か | 「スチームコンベクションオーブン／調理品質の標準化とリピーター確保」 |
| 経費内訳 | 単価×数量 | 「2,248,000×1」 |
| 補助対象経費 | 金額（税抜or税込） | 2,248,000 |

### 費目一覧と使い分け

| 費目 | 用途 | 注意点 |
|------|------|--------|
| ①機械装置等費 | 機器・設備の購入 | 汎用性の高いもの（PC等）は不可 |
| ②広報費 | チラシ・看板・広告 | |
| ③ウェブサイト関連費 | HP制作・Web広告 | **上限: 補助金総額の1/4かつ最大50万円、単独申請不可** |
| ④展示会等出展費 | 展示会・商談会 | |
| ⑤旅費 | 出張交通費 | |
| ⑥新商品開発費 | 試作・開発 | |
| ⑦借料 | レンタル・リース | |
| ⑧委託・外注費 | 工事・制作委託 | HP制作は③に計上 |

## Ⅲ. 資金調達方法

### 構造

| 左側（調達一覧） | 右側（補助金の手当方法） |
|-----------------|----------------------|
| 1. 自己資金 | 2-1. 自己資金 |
| 2. 持続化補助金 | 2-2. 金融機関からの借入金 |
| 3. 金融機関からの借入金 | 2-3. その他 |
| 4. その他 | |
| 5. 合計額 | |

### ルール
- 左側「2. 持続化補助金」= Ⅱの「(6)補助金交付申請額合計」と一致
- 左側「5. 合計額」= Ⅱの「(5)補助対象経費合計」と一致
- 右側は「補助金が交付されるまでの立替資金」の調達方法

### 実例

**コナミル（自己資金で全額立替）**
| 区分 | 金額 |
|------|------|
| 自己資金 | 1,042,000円 |
| 持続化補助金 | 2,000,000円 |
| 合計 | 3,042,000円 |

## 経費明細の書き方テクニック

### 1. 「内容・必要理由」の書き方

**悪い例:**
> スチームコンベクションオーブン

**良い例:**
> スチームコンベクションオーブン／窯の立上げ時間の削減による業務効率化効果の実現やマフィンやスコーンといった新メニューの追加

**ポイント:**
- 商品名だけでなく、**なぜ必要か**を必ず書く
- 様式2の「経営課題」「取組内容」と対応させる
- 「生産性向上」「販路開拓」のキーワードを含める

### 2. 経費内訳の書き方

```
単価×数量＝合計金額
例: 2,248,000×1
例: 1,800,000×1=1,800,000
```

### 3. 税抜・税込の選択

| 事業者区分 | 選択 |
|-----------|------|
| 課税事業者 | 税抜 |
| 免税事業者 | 税込 |
| 簡易課税事業者 | 税込 |
| 2割特例事業者 | 税込 |

### 4. 補助金額の計算

```
補助金額 = 補助対象経費 × 2/3（円未満切捨て）
ただし上限あり:
- 通常枠: 50万円（インボイス特例で+50万＝100万円）
- 創業型: 200万円（インボイス特例で+50万＝250万円）
- 賃金引上げ特例: +150万円
```

### 5. ウェブサイト関連費の制約

- 補助金交付申請額合計の **1/4以内**（最大50万円）
- ウェブサイト関連費**のみ**での申請は不可
- HP制作は③ウェブサイト関連費、制作委託も③
