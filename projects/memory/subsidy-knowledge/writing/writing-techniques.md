# 採択されるためのライティングテクニック集

最終更新: 2026-03-25
作成者: knowledge-builder
根拠: LEIMEI（広告代理店）、まりぼう（居酒屋）の採択事例分析

---

## 概要

小規模事業者持続化補助金の採択率を高めるための、実証済みライティングテクニックをまとめたリファレンス。
「伝わる書き方」「数値の使い方」「審査員が評価するポイント」を体系化。

---

## 1. 全体構成の鉄則: 課題→取組→効果の一貫性

採択される申請書の最大の特徴は**ストーリーの一貫性**。

```
[経営課題]  →  [補助事業の取組]  →  [期待される効果]
  Table 7         Table 8            Table 8
  1-3章           取組内容            効果測定
```

**チェック方法:**
Table 7の「1-3. 経営課題」で書いた課題が、Table 8の「取組内容」で解決され、「効果」につながっているか確認する。
この流れが途切れている申請書は採択されにくい。

**悪い例（課題と取組が乖離）:**
> 課題: 「認知度が低い」
> 取組: 「店舗の設備を新しくする」
> 効果: 「売上が上がる」

**良い例（一貫した流れ）:**
> 課題: 「現店舗が分かりづらい場所にあり新規顧客が来にくい」
> 取組: 「認知度の高い繁華街へ移転・内装工事」
> 効果: 「通行人の目に触れる機会が増え新規顧客○%増を見込む」

## 2. 数値の使い方

### 2-1. 数値を入れるべき箇所（必須）

| 箇所 | 入れるべき数値 | 例 |
|------|-------------|-----|
| 売上の現状 | 金額・前年比 | 「年間売上○○万円（前年比16.8%増）」 |
| 市場データ | 市場規模・成長率 | 「総広告費7兆6,730億円」 |
| 顧客規模 | 月間客数・客単価 | 「月間顧客数○名、客単価4,000円」 |
| 補助後の目標 | 売上・客数の変化 | 「客単価4,500円、年間1,330万円を目標」 |
| 設備・規模 | 面積・席数・数量 | 「20席→25席に拡大」 |

### 2-2. 前年比の計算と表現

**必ず前年比を計算して記載する。**

```
前年比 = (今年度売上 ÷ 前年度売上) × 100 - 100
例: (1,100万円 ÷ 940万円) × 100 - 100 = 17.0%増
```

表現方法:
- 「○○万円（前年比○%増）」
- 「前年度比○%の増収」
- 「○○万円→○○万円へと成長（+○%）」

### 2-3. 数値に出典を付ける

市場データには必ず出典を付ける。なければ信頼性が下がる。

**出典の書き方:**
- 「電通『日本の広告費2024』によると」
- 「厚生労働省○○調査（2023年）では」
- 「○○商工会議所の調査によると」

**使える公的データソース:**

| 業種・テーマ | データソース |
|------------|------------|
| 広告業 | 電通「日本の広告費」（毎年2月発表） |
| 飲食業 | 農林水産省「食料需給表」、総務省「家計調査」 |
| IT・Web | 総務省「情報通信白書」 |
| 小売業 | 経済産業省「商業動態統計」 |
| 全業種 | 中小企業庁「中小企業白書」 |
| 地域情報 | 各市区町村統計、RESAS（地域経済分析システム） |

## 3. 市場分析の書き方

### 3-1. マクロ→ミクロの2段階構成

採択事例（まりぼう）が示した最良パターン:

```
Step1: 全国・業界全体のデータで「市場が拡大/変化している」を示す
  ↓
Step2: 地域・自社のデータで「地元でも同様の傾向がある」を示す
  ↓
Step3: だから「自社事業に追い風がある」と結論づける
```

**テンプレート:**
```
[業種]市場は全国的に[傾向]しており、[出典]によると[具体的な数値]となっています。
[地域名]においても同様の傾向が見られ、[地域の状況を説明]。
このような市場環境の変化は、当社にとって[チャンス/課題]であり、
[補助事業の取組]によって[機会を活かせる/課題を克服できる]と考えます。
```

### 3-2. 競合分析の書き方（LEIMEI事例）

競合を具体的に挙げることで、差別化が明確になる。

**書き方のルール:**
- 同業者2〜3社を具体的に名前・資本金・特徴で比較
- 自社との違いを明確に
- 「競合にはできないが自社にはできること」を強調

**テンプレート:**
```
【競合分析】
競合A社（資本金○○万円）: [サービス内容・特徴]
競合B社（資本金○○万円）: [サービス内容・特徴]

当社との差別化ポイント:
① [差別化1: 競合にはない自社の強み]
② [差別化2]
```

## 4. SWOT分析の書き方

### 4-1. 強みの書き方

**強みは「なぜ強みなのか」の理由まで書く。**

悪い例: 「豊富な経験があること」
良い例: 「創業○年で○○業界に特化した経験を持ち、同業他社が対応困難な○○領域でも対応実績がある」

**強みの発見フレームワーク:**
- 競合他社が持っていないもの
- 顧客から特に感謝・評価されること
- 代表者・スタッフの専門性・資格・経験
- 地域における希少性（○○市で唯一の〜）
- 価格・スピード・品質での優位性

### 4-2. 弱みの書き方

**弱みは「補助事業で克服できるもの」を選ぶ。**

弱みの選び方:
- NG: 補助事業と無関係な弱み（「資金力が弱い」など）
- OK: 補助事業で直接解決できる弱み（「HPが古くて集客できていない」→HP改修で解決）

**テンプレート:**
```
① [弱み1]: [現状の問題] → [補助事業でどう解決するか]
② [弱み2]: [現状の問題] → [補助事業でどう解決するか]
```

## 5. フェルミ推定による効果試算

採択事例（まりぼう）で使われた、最も審査員に評価される効果の書き方。

### 5-1. 基本構造

```
[1単位あたりの数値] × [単位数] = [合計]
     ↑具体的              ↑根拠を添える
```

**飲食店の例:**
```
1日あたり売上 = 来客数 × 客単価
             = 11.3人 × 4,500円
             = 50,850円

月間売上 = 50,850円 × 26日（営業日）
        = 1,322,100円

年間売上 = 1,322,100円 × 12ヶ月
        ≒ 1,587万円（目標）
```

**サービス業の例:**
```
月間受注件数 = 問い合わせ数 × 成約率
           = 10件 × 40%
           = 4件

月間売上 = 4件 × 平均単価50万円
        = 200万円

年間売上 = 200万円 × 12ヶ月
        = 2,400万円（目標）
```

### 5-2. 根拠の示し方

各数値に根拠を1行添えると説得力が増す。

```
来客数11.3人の根拠:
「現在の来客数は平均8.7人/日。移転により立地条件が改善し、
通行量調査（○○交差点1日○○人）から新規来客30%増を見込む。
8.7人 × 1.3 = 11.3人。」
```

### 5-3. 現状と目標の対比表

| 指標 | 現在 | 補助事業後（目標） | 増加率 |
|------|------|-----------------|-------|
| 月間客数 | ○○人 | ○○人 | +○% |
| 客単価 | ○,○○○円 | ○,○○○円 | +○% |
| 月間売上 | ○○万円 | ○○万円 | +○% |
| 年間売上 | ○○○万円 | ○○○万円 | +○% |

## 6. 顧客の声の引用テクニック

### 6-1. なぜ顧客の声が効果的か

- 抽象的なニーズ分析より、リアルな声は審査員の共感を得やすい
- 「顧客が実際にそう言っている」という客観性
- 補助事業の必要性を第三者が証明してくれる

### 6-2. 引用の書き方

**直接引用形式（最も効果的）:**
> お客様より「場所が分かりづらくて迷ってしまった」「もっと広い席でゆっくり食べたい」といったご意見を複数いただいています。

**間接引用形式（声が集められない場合）:**
> 多くのお客様から、アクセスのしづらさについてお声をいただいており、改善を求められています。

### 6-3. 声を集める方法

- アンケート（GoogleフォームをQRで配布）
- SNSのDM・コメント
- Googleビジネスプロフィールのレビュー
- 口頭でいただいた意見をメモ

## 7. 物語性（ナラティブ）の活用

採択事例（まりぼう）が示した、審査員の心を動かす書き方。

### 7-1. 創業ストーリーの型

```
[代表者の個人的経験・背景]
  ↓
[その経験から生まれた想い・使命感]
  ↓
[事業を始めた理由]
  ↓
[今の事業への接続]
```

まりぼうの例:
> バックパッカーとして東南アジアを旅した際に現地の家庭料理に魅了され（経験）→ 帰国後も日本でその味を再現したいと独学で習得（想い）→ 天童市で居酒屋を開業（創業）→ 地元でも多様な食文化を楽しんでほしいという想いで現在も運営（現在）

### 7-2. ナラティブが効果的な理由

- 審査員も人間。感情に訴えると記憶に残る
- 事業の必然性・本気度が伝わる
- 他の申請書との差別化になる

### 7-3. 使えるシーン

- 1-1. 事業概要・沿革での創業ストーリー
- 4-1. 経営方針での「目指す姿」
- Table 8の「背景・目的」

## 8. 見出し・レイアウトのテクニック

### 8-1. 効果的な見出しの使い方

採択事例（LEIMEI）が多用していた手法:

```
【ポイント】箇条書きでまとめた重要事項
【今後の予定】時系列での計画

■ 項目名（中見出し）
● 箇条書き（小項目）
```

### 8-2. 読みやすさのルール

- 1段落は4〜5行を目安に改行
- 箇条書きは3〜5項目まで
- 数値・固有名詞は太字（審査員の目に入りやすい）
- 表（テーブル形式）を積極的に活用

### 8-3. やってはいけない書き方

- NG: 1つの段落に5つ以上の情報を詰め込む
- NG: 根拠のない形容詞（「圧倒的な」「素晴らしい」）
- NG: 補助事業との関連が不明な情報
- NG: 目標数値の根拠説明なし

---

## 9. 補助事業完了後のビジョンを書く

審査員は「補助金を渡した後にこの事業者が成長するか」を見ている。

**テンプレート:**
```
補助事業完了後は、[短期的な効果]を足がかりに、[中期的な展開]を目指します。
具体的には、[○年以内に〜を実現する計画]です。
また、[補助事業で得た知見・資産]を活かし、[さらなる事業拡大の方向性]にも
取り組んでいく予定です。
```

## チェックリスト: 採択率を高めるための最終確認

### ストーリーの一貫性
- [ ] 課題→取組→効果が一本の線でつながっているか
- [ ] Table 7の課題がTable 8の取組で解決されているか

### 数値・根拠
- [ ] 売上推移に前年比を計算して記載したか
- [ ] 市場データに出典を付けたか
- [ ] 効果試算がフェルミ推定形式で計算されているか
- [ ] 目標数値に根拠を添えたか

### 読みやすさ
- [ ] 顧客の声（実際の言葉）を引用したか
- [ ] 強みが具体的で差別化が伝わるか
- [ ] スケジュールが月単位で記載されているか
- [ ] 定性効果と定量効果を分けて書いたか

### 物語性
- [ ] 創業ストーリーや代表者の想いが伝わるか（特に飲食・サービス業）
- [ ] 補助事業後のビジョンまで書いているか
